About Dionysus

Never run out of beer again

Dionysus is the premier tool to make sure your inventory never runs out.

Features include:

  • Automatic alerting when item count drops below a pre-defined threshold
  • Set who is on call to fix the lack of items problem
  • See the latest picture taken of your setup
  • Get the estimated count of visible beers
  • Add to a refill request list to get only the best beers

Components:

  • Linear SVM classifier using HOG features
  • Go server with exposed endpoints to query statuses
  • Amazing dashboard to view consolidated information
